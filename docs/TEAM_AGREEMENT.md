## GitHub Workflow

```
Issue -> Branch -> Pull Request -> Merge -> Evidence
```

**Rules:**
- No work without Issue
- Code in branches
- Merge through PRs
- Review by team member

---

## Evidence Rule

> If it isn't linked, it didn't happen.

Each member posts 2-3 receipts weekly:
- Pull request
- Issue closed
- Code commit
- Review comment
- Documentation update

Post receipts as comments on Weekly Sprint Packet issue.
